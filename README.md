# SQL + PySpark Interview Prep

A focused 4-week sprint to build product-company-level depth in SQL and PySpark, 
alongside Databricks Data Engineer Associate certification prep.

## Why this repo exists
I'm a Data Engineer (2 yrs DE, 4 yrs total) with production experience in PySpark, 
Delta Lake, and Databricks on Azure. This repo documents a structured effort to close 
the gap between "comfortable with daily tooling" and "interview-ready at a product 
company" — specifically around SQL query optimization and PySpark performance tuning.

## Structure
- `/sql` — SQL problems and optimization exercises, with EXPLAIN ANALYZE breakdowns
- `/pyspark` — PySpark scripts covering joins, window functions, partitioning, and 
  performance debugging
- `/notes` — Concept notes in my own words (architecture, shuffles, caching, etc.)
- `/mock-interviews` — Timed practice sessions and post-mortems
